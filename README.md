# 🎵 MP3 Music Player

> *Your personal music companion — built with Python, powered by passion.*

A lightweight yet feature-packed **MP3 Music Player** built with Python. Whether you want to vibe to one song on repeat or cruise through your entire playlist, this player has you covered.

---

## ✨ Features

| Feature | Description |
|--------|-------------|
| ▶️ **Play** | Instantly load and play any MP3 file |
| 🔁 **Loop Mode** | Keep your favorite track playing on repeat |
| 🔄 **Replay** | Restart the current song from the beginning |
| ⏭️ **Next Song** | Skip to the next track in your playlist |
| ⏸️ **Pause & Resume** | Pause anytime and pick up right where you left off |
| ⏹️ **Stop** | Stop playback instantly |
| 🎶 **Playlist Support** | Queue multiple songs and enjoy non-stop music |

## 🧠 How It Works

1. On launch, the player **loads all MP3 files** from the `Music/` folder
2. Songs are added to a **playlist queue** automatically
3. You can **skip, replay, or loop** any song using keyboard controls
4. **Loop mode** keeps the current song replaying until toggled off
5. **Next song** moves to the next track in the queue and loops back to the start
